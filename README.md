# Simulasi CPNS

Simulasi CPNS is a mobile application built with .NET MAUI.  
This app helps users practice CPNS-style questions and improve their preparation.

## Features

- Practice CPNS questions
- View question choices
- Show score/result
- Simple and clean mobile UI

## Tech Stack

- .NET MAUI
- C#
- XAML
- SQLite

## Color Theme

This app uses a motivational red theme to create a more energetic and encouraging learning atmosphere.

- Primary Red: `#E11D48`
- Dark Red: `#BE123C`
- Accent Pink Red: `#FB7185`
- Soft Background: `#FFF1F2`
- Soft Surface: `#FFE4E6`
- Main Text: `#1F2937`
